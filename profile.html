ABOUT

Stembapost Website Application


Stembapost merupakan aplikasi lokal buatan salah satu team beranggotakan 2 anggota girls , siswa SMKN 7 Semarang(STM Prmbangunan). Aplikasi ini merupakan aplikasi lokal berbasis web yang dibangun dengan interface Framework CodeIgniter, Bootstrap 3, jQuery, HTML 5 dan CSS-Saas. Untuk backend nya digunakan DBMS MySQL(PHPMyAdmin) dan Restfull API Node JS sebagai connector dengan API Whatsapp.
Cara Kerja Aplikasi ini ialah user akan membuat pesan Broadcast dari laman website stembapost dengan memasukkan nomor dan pesan lalu klik send button. Selanjutnya pesan tersebut akan otomatis terbroadcast langsung ke nomor Whatsapp yang dituju.

IMAGES & FEATURES

Login Page

Input Username dan Password di halaman ini.

Sign Up Page

Lakukan sign up jika belum memiliki akun, isi previlleges yang disediakan seperti super user, admin, atau user. Lalu isi Full Name dan Username serta password.

Dashboard Menu

Berisi ringkasan pesan , contact dan detail lain yang ditampilkan dalam box-div. Terdapat pula History List untuk menampilkan history message yang telah berhasil terbroadcast

List Contacts Menu

List Contact menampilkan data kontak yang tersimpan dalam database , terdapat fitur import contact untuk mengimpor seluruh kontak yang tersimpan.

Address List Menu

Tak jauh berbeda dengan list contact, address list menu membantu user untuk memudahkan pencarian kontak berdasarkan beberapa kategori. Kategori tersebut berupa kelas, jurusan dan sebaginya.

New Broadcast Menu

Menu main dari aplikasi stembapost ialah New Broadcast. Menu ini memungkinkan user untuk melakukan interaksi dan broadcast kepada kontak lain.

Trash Menu

Trash Menu yaitu menu untuk menyimpan pesan yang dihapus namun belum terhapus secara permanen. Dalam menu ini, user dapat menghapus pesan terkirim secara permanen.

Set Up Menu

Set Up Menu merupakan menu untuk mengatur profile, kontak, dan sebagainya.

About Menu

Berisi detail developer dan detail informasi mengenai aplikasi stembapost
